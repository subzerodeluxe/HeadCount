# HeadCount

HeadCount levert de student een nuttige tool waarmee hij inzicht krijgt in z’n aanwezigheid en daaraan gekoppelde prestaties. Daarnaast levert de app ondersteuning om vervelende patronen te doorbreken en vooraf vastgestelde doelen te behalen. Bovenal geeft het studenten en docenten de kans op een betere samenwerking met als hoofddoel: de productiviteit van een student te verbeteren.
